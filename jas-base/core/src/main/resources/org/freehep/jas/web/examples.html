Most of these examples can also be run in batch mode by using JAIDA directly (see release notes at http://java.freehep.org/jaida ).

Many packages of the DataMelt computational platform are also available. You can use Java classes from the "jhplot" packages, such as canvases (HPlot, HPlot3D), data containers (P0D, P1D, etc.) and histograms (H1D, H2D, etc.).
