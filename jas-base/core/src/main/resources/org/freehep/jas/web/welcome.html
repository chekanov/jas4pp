JAS4PP

Java Analysis Studio for Particle Physics

version 1.5 (June 2020). Based on JAS 3.1.4, LCSIM 4.0 and LCIO 2.7.4

JAS4PP is an AIDA compliant data analysis program for particle physics based on Java Analysis Studio, JAS3. In addition to the usual JAS3 functionality, JAS4PP includes several enhancements needed for particle collision experiments. See the JAS4PP web site for the list of features.
