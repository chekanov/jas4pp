JAS4PP examples

These examples are written using the Groovy scripting language. To open an example click on the hyperlink below, then choose "Run" from the editor's pop-up menu to run them. For more information see the AIDA User's Guide.

CreateAndPlotHistograms.groovy Simple example of creating histograms and plotting them.
Fit.groovy Fits a plot using two different fitters, and compares the result.
CreateAndFitDataPointSet.groovy Creates an IDataPointSet and then fits it.
HistogramArithmetic.groovy Examples of performing histogram arithmetic
Cloud.groovy Simple use of "Clouds" (unbinned histograms)
Tuple.groovy An example of creating and manipluating an NTuple
ComplexFit.groovy A more complicated fitting example
ScansAndContour.groovy Create scans and contours
AccessDataOnMasterTree.groovy This example shows how to access data on the master tree.
Style1D.groovy Simple example to set basic styles for 1D plots.
Style2D.groovy Simple example to set basic styles for 2D plots.
Styles.groovy This example gives an overview of the available styles on a plot.

histo1.groovy Creating histograms and plotting them.
histo2.groovy Another 1D histogram example.
histo2D.groovy Creating 2D histogram.
graph.groovy Graphs with points using GROOT.
func_1D.groovy Creating 1D function.
func_2D.groovy Creating a 2D function.
histo2D_2hf.groovy Creating a histogram and a function in 3D.
points0.groovy Data points.
data_p1d_1.groovy Data points in X-Y (errors can be shown).
